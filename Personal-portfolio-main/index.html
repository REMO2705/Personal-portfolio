These early languages provided me with a solid understanding of fundamental programming concepts, logic, and syntax. As my proficiency in these languages grew, I gradually transitioned towards web development technologies, immersing myself in the dynamic trio of HTML, CSS, and JavaScript. This shift opened up a new realm of possibilities as I started crafting interactive and visually appealing web pages. As I honed my skills in frontend development, I embarked on the exciting journey of implementing various projects, applying my newfound knowledge to real-world scenarios. This phase marked the beginning of my exploration into the vast and ever-evolving landscape of web development, laying the groundwork for future endeavors and a continuous pursuit of learning and growt

2021

In 2021, I crafted fundamental front-end projects, refining skills in HTML, CSS, and JavaScript. These ventures laid the groundwork for exploring more advanced technologies like Next.js, enhancing my ability to build robust and optimized React applications. In 2023, I expanded my skills by learning React, a powerful JavaScript library for building interactive user interfaces. Simultaneously, I delved into Database Management Systems (DBMS), enhancing my ability to organize and manage data effectively.
